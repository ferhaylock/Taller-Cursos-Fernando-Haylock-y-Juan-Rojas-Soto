Curso de angular

Aprenderás a crear y mantener aplicaciones web en una sola página con Angular. A través de este framework para aplicaciones web, trabajarás con HTML y TypeScript, pudiendo generar desde el prototipo hasta la implementación completa de una aplicación.

Contenido del curso

  • Introducción al curso y a Angular
  • Componentes y Elementos de un proyecto Angular
  • Typescript
  • Interpolación y Directivas
  • Comunicación entre componentes
  • Formularios en Angular / Reactive Forms
  • Angular Material
  • Pipes y Directivas personalizadas
  • Servicios y RxJS
  • Introducción a la programación reactiva con RXJS
  • Router
  • Lazy Loading de módulos y Guards
